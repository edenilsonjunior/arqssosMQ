# Controle de Temperatura com Fila de Mensagens

Este projeto em Java é uma implementação manual de uma fila de mensagens para monitoramento e controle da temperatura em vários cômodos. A aplicação permite a inserção e o processamento de mensagens relacionadas a leituras de temperatura, com a fila gerenciando a ordem e a prioridade das mensagens.

## Descrição

O sistema é projetado para representar uma fila de mensagens que armazena e processa dados de temperatura provenientes de diferentes cômodos. Ele simula o controle de temperatura, permitindo a adição de mensagens na fila e o processamento dessas mensagens para realizar ações de controle, como ajustes de temperatura ou geração de relatórios.

### Funcionalidades

- **Inserção de Mensagens:** Adicione novas mensagens na fila com dados de temperatura e informações sobre o cômodo.
- **Processamento de Mensagens:** Consome mensagens da fila e realiza ações com base nas informações de temperatura.
- **Gerenciamento de Fila:** A fila é gerenciada de forma a manter a ordem correta das mensagens e garantir o processamento adequado.
